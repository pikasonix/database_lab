## Installation

Install NodeJS v20.12.2
https://nodejs.org/en/download/prebuilt-installer
Install npm
```bash
  npm install
```
    
## Usage
Run project
```bash
npm start
```

Run to convert SCSS to CSS
```bash
npm run watch
```
